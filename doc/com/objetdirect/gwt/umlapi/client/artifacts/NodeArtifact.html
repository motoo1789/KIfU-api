ClassArtifact, ObjectArtifact
public abstract class NodeArtifact
extends BoxArtifact

This class is an artifact used to represent a class.
A class is divided in three NodePartArtifact :

  • ClassPartNameArtifact For the name and stereotype part
  • ClassPartAttributesArtifact For the attribute list part
  • ClassPartMethodsArtifact For the method list part
