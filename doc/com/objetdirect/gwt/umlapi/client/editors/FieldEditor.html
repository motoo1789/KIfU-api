public abstract class FieldEditor
extends Object

This abstract class is a generic field editor for uml artifacts.
It displays a TextBox and update the artifact if the user hit Enter or the TextBox lose the focus
or cancel if the user hit Escape
It also supports multiple lines edition with TextArea
